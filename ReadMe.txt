Below are instructions to publish this to the cloud.

Table is currently hosted @ https://wagglygomersall.azurewebsites.net

1) Code is fully set to be published using Visual Studio's Azure App Publish function. The application is currently hosted on "WagglyGomersall" App Services through Azure
2) The default connection string is established in the code to connect to a Azure SQL Server "wagglygomersall" and Database "Waggly". The resource group is "DogPage" The code requires the database be a dependency.
